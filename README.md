# 🛁 Customer Experience Analytics - Personal Care Products Case Study

## 📋 Overview
This project presents a case study analyzing customer experience with personal care products on a website. Data was collected via a questionnaire, stored in a CSV file, and analyzed using Python.


## 🛠️ Technologies Used
- **Backend**: Python, Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQLite/MySQL
- **Tools**: Pandas, NumPy for statistical analysis

## 📦 Requirements
- Python 3.8 or newer
- pip (package manager)

## 🚀 Installation and Running

### 1. Clone the Repository
\`\`\`bash
git clone https://github.com/khadija-saifeldin/Website-Customer_Experience_Analytics_Case_Study_Personal_Care-Products.git
cd Website-Customer_Experience_Analytics_Case_Study_Personal_Care-Products
\`\`\`

### 2. Install Required Libraries
\`\`\`bash
pip install -r requirements.txt
\`\`\`

### 3. Run the Application
\`\`\`bash
python app.py
\`\`\`

Then open your browser and navigate to: \`http://localhost:5000\`
